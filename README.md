# D2D Expense Tracker

A web-based expense tracker built using ASP.NET MVC, ADO.NET, and SQL Server.

## Features
- Track daily and monthly expenses and income
- Investment tracking (SIPs)
- Savings calculation
- PDF report generation
- Login/registration system

## Tech Stack
- ASP.NET MVC 
- SQL Server
- ADO.NET
- HTML/CSS/JS
